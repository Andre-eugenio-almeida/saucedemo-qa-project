# SauceDemo QA Project

## 📌 Sobre o Projeto

Este projeto foi desenvolvido para demonstrar conhecimentos em Qualidade de Software (QA), aplicando boas práticas utilizadas em projetos reais.

O objetivo é validar as funcionalidades do sistema SauceDemo por meio de testes manuais e automatizados, utilizando uma abordagem estruturada que inclui análise de requisitos, planejamento de testes, criação de casos de teste, execução, registro de defeitos e automação.

O projeto será desenvolvido em etapas, simulando o fluxo de trabalho de uma equipe de QA em um ambiente ágil.

---

## 🎯 Objetivos

* Realizar análise de requisitos.
* Elaborar plano e estratégia de testes.
* Criar casos de teste manuais.
* Executar testes funcionais.
* Registrar defeitos encontrados.
* Automatizar testes Web.
* Automatizar testes de API.
* Implementar integração contínua (CI/CD).
* Gerar relatórios de execução.
