


# 📌 Sobre o Projeto

O **SauceDemo QA Project** é um projeto desenvolvido para demonstrar conhecimentos em **Qualidade de Software (QA)**, aplicando boas práticas utilizadas em ambientes profissionais.

O projeto simula o fluxo de trabalho de uma equipe de QA, abrangendo todas as etapas do processo de testes:

- Levantamento e análise de requisitos;
- Planejamento e estratégia de testes;
- Elaboração de casos de teste;
- Execução de testes manuais;
- Registro e acompanhamento de defeitos;
- Automação de testes Web;
- Automação de testes de API;
- Integração Contínua (CI/CD).

O objetivo é construir um portfólio completo, demonstrando competências em testes manuais e automatizados utilizando a aplicação **SauceDemo**.
